Why This Page

브랜드 없는 기본 화면 대신 바로 시연 가능한 인상

색, 배경, 타이포, 상태 카드가 이미 들어가 있어서 Flask 앱을 띄우자마자 제품 데모처럼 보이도록 구성했습니다.

Template Ready

Jinja 템플릿 기반으로 바로 확장 가능한 구조

현재는 정적 샘플이지만, 이후 `rooms`, `metrics`, `messages` 같은 데이터를 주입해서 운영 화면으로 바꾸기 쉬운 형태입니다.

Responsive

데스크톱과 모바일 모두 무리 없이 읽히는 레이아웃

큰 카드 중심의 그리드에서 작은 화면에서는 단일 컬럼으로 자연스럽게 전환되도록 브레이크포인트를 잡았습니다.

Flask의 기본 `render_template("sample.html")` 연결만으로도 제품 분위기를 확인할 수 있는 샘플 페이지입니다. 필요하면 다음 단계에서 실제 채팅방 목록과 대화 이력을 주입하면 됩니다.
