Signal Room

Cube 챗봇 운영 페이지

이 랜딩 페이지에서 현재 열어둘 수 있는 운영 화면으로 바로 이동할 수 있습니다. 대화 이력 확인, 서비스 상태 점검, 스케줄 작업 추적, 파일 전달, 워크플로우 시각화를 한 곳에서 시작합니다.

Available Pages

Debug /conversation

Conversation History

최근 저장된 대화 메시지를 확인하고 사용자별 메시지 흐름을 빠르게 점검합니다.
Ops /monitor

Service Monitor

MongoDB, Redis 기반 기능, worker heartbeat 상태를 주기적으로 다시 확인합니다.
Jobs /scheduled_tasks

Scheduled Tasks

등록된 작업 목록, 마지막 실행 상태, 최근 활동 이력을 한 화면에서 추적합니다.
Files /file_delivery

File Delivery

업로드된 파일 목록을 확인하고 공유 가능한 URL 기반 전달 기능을 테스트합니다.
Graph /workflows

Workflow Graphs

등록된 워크플로우 목록으로 이동한 뒤 개별 그래프를 열어 구조를 검토합니다.

운영자가 가장 자주 보는 화면은 대화 이력과 모니터링 화면입니다. 필요하면 이 랜딩 페이지를 기본 진입점으로 유지한 채 다른 디버그 페이지를 계속 추가할 수 있습니다.
