# LLM Chatbot

## Python Version

This project is based on Python 3.11.

- Local version file: `.python-version` (`3.11`)
- Runtime hint file: `runtime.txt` (`python-3.11.11`)

## Quick Start

```bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python index.py
```
